## Virtual Spatial Augmented Body (V-SAB)プラットフォームとは
　Virtual Spatial Augmented BodyプラットフォームはVR空間内でサイズ変容したアバターを用いた心理・認知実験を実施できるプラットフォームです。
アバターはVRMモデルのボーンを適応したVRoid, Daz3D, MakeHumanモデルが使用できます．
